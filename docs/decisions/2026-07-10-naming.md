# 2026-07-10 — Naming

Status: **decided — Okolo** (okolo.events), confirmed by George 2026-07-14 · Owner: George

## Final name
**Okolo** (Slavic "around/nearby" — same conceptual fit as Umkreis, but open vowels and globally
pronounceable). Live at okolo.events; shipped UI/i18n/meta already use it. Remaining legacy strings
deliberately NOT renamed: the Postgres schema `umkreis`, the `umkreis_droppin_hint` localStorage
key, and the crawler UA `UmkreisBot` (outward-facing — rename is a separate decision).

## Working name (historical)
**Umkreis** (German "radius / vicinity" — literal fit). `umkreis.at`, `umkreis.events`, `umkreis.io`
were free; `umkreis.com` parked/expensive (~$4.7k). Used in UI + `llms.txt` until the rename.

## Direction
George wants a **catchy, international, easily pronounced worldwide** name on a **`.events`** TLD;
can be an English word, remote synonym, nerd reference, or a German/Nordic/Slavic word. Worldwide
ambition, so favor open vowels, no umlauts, no tricky clusters.

## Candidates checked (all `.events`, free unless noted)
**Favorites (short, globally sayable, memorable):**
- `grok.events` — Heinlein/dev-culture "to deeply understand"; short, cool, brandable.
- `afoot.events` — "something's afoot" = something is happening (exact meaning in one idiom).
- `ambit.events` — the range/scope around a point (elegant conceptual fit).
- `sidequest.events` — gaming: life's main quest + local side quests; playful, self-explanatory.
- `nabo.events` — Danish/Norwegian "neighbor"; warm, 2 syllables, easy everywhere.
- `okolo.events` — Slavic "around/nearby"; phonetically bulletproof (o-ko-lo).
- `respawn.events` — gaming energy; events respawn.
- `whereabouts.events` / `hereabouts.events` — literal "where things are / around here".

**Also free:** parsec, quadrant, umbra, orbiter, waypoints, althing (Norse assembly), hither,
roundabout, homebase, haps, goingson, outings, doings, powwow, wingding, jaunt, lark, caper, romp,
haunts, blisko, skupa, sammen.

**Premium-priced (heads up):** ping (~$99/yr — great fit), hangout ($200), shindig ($66), gig/scout ($33).
**Taken/aftermarket:** kolo, lumo, iskra, orbit, pulse, vibe, tempo, nova, swarm, nearby.

Lean **grok.events** (brandability) / **afoot.events** (literal meaning) / **sidequest.events**
(young + playful) among the English/nerd set, or **nabo.events** / **okolo.events** for a warm,
neutral-international word. Pending: George picks → verify trademark → register → rebrand UI +
metadata + `llms.txt` (a ~1-hour task, tracked in `tasks/todo.md`).
